# Dismissed contradiction: homogeneous isotropic FCQPT model vs material-specific RBC/Hall calculation

## Candidate tension

- Existing parent context: the YbRh2Si2 FCQPT branch includes a premise that models YbRh2Si2 and related heavy-fermion compounds as a homogeneous isotropic heavy-electron liquid, deliberately neglecting crystal-lattice anisotropy, Brillouin-zone structure, multiple bands, and anisotropic effective masses for universal scaling.
- New selected root: Friedemann et al. 2010 use a material-specific renormalized-band calculation with experimental CEF splittings, a thermodynamically calibrated 4f quasiparticle resonance width, band-resolved Hall transport products, and explicit YbRh2Si2/YbIr2Si2 DOS/gamma values.

## Verdict

Dismissed as a scope/suspicion note, not promoted to `contradiction()` or `equivalence()`.

## Rationale

The two claims can both be true: a homogeneous isotropic FCQPT model can be used as a universal scaling approximation while a renormalized-band calculation provides material-specific band/Hall/DOS evidence that the approximation omits. The extension qualifies the parent premise's scope, but it does not logically exclude the parent model's stated universal-scaling use and it is not the same proposition as the FCQPT scheme.

## Parent synthesis action

The parent package adds `cross_ybrh2si2_rbc_qualifies_homogeneous_isotropic_scope`, supported by the existing homogeneous-isotropic premise and the Friedemann2010 RBC root/helpers.
