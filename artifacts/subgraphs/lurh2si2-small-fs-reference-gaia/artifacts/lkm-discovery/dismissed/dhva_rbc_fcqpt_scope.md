# Dismissed contradictions: LuRh2Si2/YbRh2Si2 dHvA reference vs existing YbRh2Si2 context

## Candidate tensions checked

- New 2013 dHvA reinterpretation root: 5-7 kT YbRh2Si2 peaks are reassigned as harmonics, leaving a distinct 14 kT fundamental whose absence from the refined core-4f small-FS calculation supports high-field itinerant Yb 4f participation.
- Existing Friedemann 2010 RBC/Hall branch: material-specific renormalized-band calculations include 4f-derived quasiparticle resonance physics, CEF/gamma calibration, and band-resolved Hall/DOS values.
- Existing Shaginyan 2010 FCQPT branch: YbRh2Si2 is modeled with a homogeneous isotropic heavy-electron liquid that deliberately neglects band topology and anisotropy to describe universal scaling of effective mass.

## Verdict

Dismissed as scope/context tensions, not promoted to `contradiction()`.

## Rationale

The 2013 small-FS reference and 2010 RBC/Hall branch are compatible rather than mutually exclusive: one uses a core-4f small-FS calculation as a negative reference for the 14 kT dHvA orbit, while the other represents itinerant 4f quasiparticles in a renormalized-band heavy-Fermi-liquid calculation. The 2013 result is also not a strict contradiction of the FCQPT branch because the homogeneous isotropic model is explicitly a universal-scaling approximation that omits material-specific band topology; the dHvA chain supplies omitted Fermi-surface details without logically excluding the scaling model.

No existing package claim asserts that the same 5-7 kT YbRh2Si2 peaks are independent fundamentals under the same data treatment, nor that the refined core-4f small-FS calculation predicts a (100)-field 14 kT orbit. Without such a same-scope incompatible claim, the audited action is dismissal plus synthesis caution.
