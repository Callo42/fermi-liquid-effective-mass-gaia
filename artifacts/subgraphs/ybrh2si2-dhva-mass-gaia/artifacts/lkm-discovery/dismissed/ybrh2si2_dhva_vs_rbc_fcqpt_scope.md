# Dismissed contradiction: YbRh2Si2 high-field dHvA evidence vs existing FCQPT/RBC context

## Candidate tension

- New selected root: Knebel et al. 2006 report high-field dHvA frequencies and cyclotron masses for YbRh2Si2 at 12-28 T with H parallel a, and show that the measured basal-plane angular dependence disagrees with one itinerant-4f LDA/FLAPW prediction set while qualitatively resembling LuRh2Si2 reference calculations.
- Existing FCQPT context: Shaginyan et al. 2010 model YbRh2Si2 with a homogeneous isotropic heavy-electron liquid for universal M*(T,B) scaling, deliberately omitting band anisotropy and multiband Fermi-surface details.
- Existing RBC/Hall context: Friedemann et al. 2010 use a material-specific renormalized-band calculation calibrated by CEF and specific heat to discuss quasiparticle bands, Hall cancellation, and DOS/gamma values.

## Verdict

Dismissed as a Gaia `contradiction()` and retained as a synthesis caution.

## Rationale

The three contexts can coexist. The dHvA result constrains high-field Fermi-surface orbits and highlights the failure of a simple itinerant-4f LDA prediction set. The FCQPT branch is a universal-scaling effective-mass approximation, not a claim about the same high-field dHvA angular frequencies. The RBC branch is a phenomenological heavy-Fermi-liquid band calculation, not the same itinerant-4f LDA calculation dismissed by Knebel et al. No pair asserts mutually exclusive values for the same observable under the same field regime and modeling assumptions.

## Synthesis obligation

When this subgraph is merged into the parent package, treat it as experimental Fermi-surface and mass evidence that qualifies model scope. Promote a contradiction only if a later selected root asserts incompatible dHvA frequencies, masses, or angular dependence for the same YbRh2Si2 field/orientation conditions.
